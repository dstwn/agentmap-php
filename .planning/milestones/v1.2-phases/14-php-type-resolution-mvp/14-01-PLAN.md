    Imports: createRequire from "node:module"; PSR4Resolver from "./PSR4Resolver.mjs". Use _require = createRequire(import.meta.url) to load tree-sitter and tree-sitter-php (CJS modules).

    Class structure mirrors PhpParser exactly: constructor sets this._parser = null, this._phpLang = null, this._ready = false. init() guard checks this._ready before loading. Single parser instance reused across all resolve() calls on same TypeResolver instance (per Pitfall 4 — performance).

    resolve(filePath, text, psr4Map, projectRoot): wrap entire body in try-catch returning { assignedTypes:[], phpDocTypes:[] } on any error. Steps: call this.init(), parse text, get rootNode, call this._collectUseImports(root), call this._extractAssignments(root, useMap, psr4Map, projectRoot), call this._extractPhpDoc(root). Return { assignedTypes, phpDocTypes }.

    _walk(node, fn): recursive pre-order walk — fn(node), then iterate node.children calling _walk recursively. Copy this from PhpParser._walk verbatim (it is the established pattern).

    _findChild(node, types): iterate node.children, return first child whose .type is in the types array. Copy from PhpParser verbatim.

    _collectUseImports(root): walk for "namespace_use_declaration" nodes. For each, iterate children looking for "namespace_use_clause". For each clause: get qualified_name child (fqcn = qn.text.replace(/^\\/,"")), check clause.childForFieldName("alias") — if present use aliasNode.text as key, else use last segment of FQCN split on "\\". Store in useMap object. Return useMap.

    _extractAssignments(root, useMap, psr4Map, projectRoot): walk for "assignment_expression" nodes. Per node: lhs = node.child(0), rhs = node.child(2). Skip if lhs.type !== "variable_name". If rhs.type === "object_creation_expression": use _findChild(rhs, ["qualified_name","name","relative_scope"]) for class node; skip if null or text is "static"/"self"/"parent" (relative_scope — skip those, no FQCN to resolve); rawName = classNode.text.replace(/^\\/,""); fqcn = useMap[rawName] || rawName; resolvedPath = (psr4Map && projectRoot) ? (new PSR4Resolver().resolve(fqcn, projectRoot, psr4Map) || null) : null; store relative path: if resolvedPath starts with projectRoot, strip prefix and leading slash. Push { type:"assigned", variable:lhs.text, className:fqcn, resolvedPath, confidence:"MEDIUM", position:node.startPosition }. If rhs.type === "scoped_call_expression": clsNode = _findChild(rhs, ["qualified_name","name"]); skip if null; rawName = clsNode.text.replace(/^\\/,""); fqcn = useMap[rawName] || rawName; methodNode = rhs.child(2) (the name after "::" ); push { type:"static_call", variable:lhs.text, class:fqcn, method:methodNode?.text || "?", confidence:"MEDIUM", position:node.startPosition }. Return results array.

    _extractPhpDoc(root): walk for "comment" nodes where node.text.startsWith("/**") (filters out // and /* per Pitfall 3). For each: row = node.endPosition.row (use endPosition per Pitfall 6); tags = this._parseDocBlock(node.text); skip if tags.length === 0. Add confidence:"MEDIUM" to each tag entry. Push all tag entries (with row field from the comment's endPosition.row) to results. Return flat results array.

    _parseDocBlock(text): apply four regex patterns exactly as in RESEARCH.md Pattern 4:
    - @var: /@var\s+([\w\\|?\[\]]+)(?:\s+(\$\w+))?/g → { tag:"@var", type:m[1], name:m[2]||null }
    - @return: /@return\s+([\w\\|?\[\]]+)/g → { tag:"@return", type:m[1], name:null }
    - @param: /@param\s+([\w\\|?\[\]]+)\s+(\$\w+)/g → { tag:"@param", type:m[1], name:m[2] }
    - @property (including -read/-write variants): /@property(?:-read|-write)?\s+([\w\\|?\[\]]+)\s+(\$\w+)/g → { tag:"@property", type:m[1], name:m[2] }
    Return flat tags array.
