  <behavior>
    - detect(root, {}, [], []) where root has no LEGACY_DIRS subdirs → returns []
    - detect(root, {}, [], []) where root has a "lib/" subdir → returns [{type: "legacy-dir", directory: "lib/", message: "..."}]
    - detect(root, {"App\\": "src/"}, [], []) where root has "src/" subdir → returns [] (src/ is PSR-4 covered, no false positive per Pitfall 5)
    - detect(root, {}, ["database/seeders/"], []) → returns [{type: "classmap", entry: "database/seeders/", message: "classmap entry: database/seeders/"}]
    - detect(root, {}, [], ["app/helpers.php"]) → returns [{type: "autoload-file", entry: "app/helpers.php", message: "autoload.files entry: app/helpers.php"}]
    - detect(root, {}, ["vendor/some/path"], []) → "vendor/" path does NOT produce a warning (Pitfall 6)
    - detect(root, {"App\\": ["app/", "src/"]}, [], []) where root has "src/" → [] (array-valued psr4 dirs correctly cover src/)
    - PSR-4 prefix trailing-slash normalization: {"App\\": "src/"} covers both "src/" and "src" candidate forms
  </behavior>
  <action>
    Create src/Core/LegacyDetector.mjs as a new ESM module. Export named class LegacyDetector.

    The class has one public method: detect(projectRoot, psr4Map, classmaps, autoFiles).

    Implementation:
    1. Build coveredDirs Set: iterate Object.values(psr4Map). For each value (string or array), normalize to array. For each dir string, add both dir.replace(/\/+$/, "") and dir.replace(/\/+$/, "") + "/" to coveredDirs (cover both "src" and "src/" forms).
    2. Initialize warnings = [].
    3. Heuristic legacy-dir check: iterate LEGACY_DIRS. For each dir, build absDir = join(projectRoot, dir). If existsSync(absDir) AND coveredDirs has neither dir.replace(/\/+$/, "") nor dir (the trailing-slash form) → push {type: "legacy-dir", directory: dir, message: `Directory '${dir}' exists but is not registered as a PSR-4 namespace prefix`}.
    4. classmap entries: for each entry of classmaps, skip if entry.startsWith("vendor/"). Push {type: "classmap", entry, message: `classmap entry: ${entry}`}.
    5. autoload-file entries: for each file of autoFiles, skip if file.startsWith("vendor/"). Push {type: "autoload-file", entry: file, message: `autoload.files entry: ${file}`}.
    6. Return warnings.

    Imports: existsSync from "node:fs"; join from "node:path"; LEGACY_DIRS from "./constants.mjs".
  </action>
