# Milestone v1.3: CI Integration Testing

**Status:** ✅ SHIPPED 2026-06-21
**Phases:** 17-19
**Total Plans:** 5
**Git range:** `4a25f5d` → `ee7061f`

## Overview

Hardened the agentmap-php CI pipeline with three surgical fixes: corrected a glob bug silently skipping 43 tests, added integration tests against the real laravel/framework fixture, and added c8-based coverage reporting across the subprocess boundary so src/Core/ modules are visible in coverage output.

## Phases

### Phase 17: CI Glob Fix

**Goal**: CI reliably runs all 256 tests on every push and PR
**Depends on**: Nothing (first v1.3 phase)
**Plans**: 1 plan

Plans:
- [x] 17-01: Fix CI test glob — change `node --test test/*.test.mjs` to `npm test`

**Details:**
Changed `.github/workflows/ci.yml` `Run tests` step to use `npm test`, delegating glob expansion to package.json which already had the correct two-glob pattern. Fixes silent skipping of `test/vue-sfc/` (43 tests). All 256 tests now run on every push/PR across Node 18/20/22 matrix.

### Phase 18: Integration Tests

**Goal**: Users can verify core CLI flags work end-to-end against a real Laravel repo in CI
**Depends on**: Phase 17 (fixed glob picks up new test file automatically)
**Plans**: 2 plans

Plans:
- [x] 18-01: Write `test/integration-laravel.test.mjs` — 4 tests covering --packages, --types, --legacy
- [x] 18-02: Add CI clone step for laravel/framework fixture (node-20 leg only)

**Details:**
Created `test/integration-laravel.test.mjs` using existing `run()`/`existsSync` helpers. Tests skip gracefully when fixture absent. CI clones `laravel/framework` on node-20 leg via `git clone --depth 1`. Full suite runs 260 tests (256 + 4 integration).

### Phase 19: Coverage Reporting

**Goal**: Coverage summary appears in CI logs on every run, capturing src/Core/ module coverage via subprocess boundary
**Depends on**: Phase 17, Phase 18
**Plans**: 2 plans

Plans:
- [x] 19-01: Add `c8@11.0.0` to devDependencies + add `coverage/` to .gitignore
- [x] 19-02: Add `Coverage report` + `Upload coverage artifact` steps to CI (node-20 conditioned)

**Details:**
Added c8 as exact-pinned devDep. CI runs `npx c8 --reporter=text --reporter=lcov npm test` on node-20 leg. c8 wraps execFileSync via NODE_V8_COVERAGE, capturing src/Core/ at 88.64% stmts. lcov.info uploaded as CI artifact. No threshold enforcement (deferred to v1.4).

## Milestone Summary

**Key Decisions:**
- Use `npm test` in CI instead of bare glob — keeps CI in sync with package.json automatically
- Scope laravel/framework clone + coverage to node-20 leg only — avoid triple-running
- Use c8 (not --experimental-test-coverage) — only tool that crosses subprocess boundary
- No coverage threshold in v1.3 — baseline 88.64% measured, enforcement deferred to v1.4

**Issues Resolved:**
- CI silently skipped 43 tests (test/vue-sfc/ not covered by glob)
- No integration tests against real PHP fixture
- No coverage visibility for src/Core/ modules

**Issues Deferred:**
- Coverage threshold enforcement (v1.4 — baseline now known: 88.64% stmts)
- Full integration suite beyond --packages/--types/--legacy (v1.4)
- Integration tests running twice on node-20 (once plain, once under c8) — optimize in v1.4

**Technical Debt Incurred:**
- Double test run on node-20 CI leg (plain npm test + c8-wrapped npm test)
