Step 1 — Coverage report:
```
      # Collect V8 coverage across subprocess boundary (node-20 leg only).
      # c8 wraps npm test, setting NODE_V8_COVERAGE so agentmap.mjs subprocess
      # inherits it. No threshold flags — CI passes regardless of %.
      - name: Coverage report
        if: matrix.node-version == 20
        run: npx c8 --reporter=text --reporter=lcov npm test
```

Step 2 — Upload coverage artifact:
```
      - name: Upload coverage artifact
        if: matrix.node-version == 20
        uses: actions/upload-artifact@v4
        with:
          name: lcov-report
          path: coverage/lcov.info
          if-no-files-found: warn
```

Do not modify any other steps. Do not remove the `Run tests` step. The coverage step runs npm test a second time (wrapped by c8) — this is intentional: the first run produces test results, the second run produces coverage data. Both are conditioned appropriately.

Note: `if: matrix.node-version == 20` uses the unquoted integer `20`, matching how the existing `Clone laravel/framework fixture` step conditions on node-20 in this file.
